title: Scope Definition
---

This CNA assigns CVE IDs for all active packages hosted on [Hex.pm](https://hex.pm/)
when no other more precise CNA is applicable. It also covers all active projects
hosted under the GitHub organizations [@erlef](https://github.com/erlef),
[@erlef-cna](https://github.com/erlef-cna), [@elixir-lang](https://github.com/elixir-lang),
[@hexpm](https://github.com/hexpm), and [@gleam-lang](https://github.com/gleam-lang).
This CNA does not assign CVE IDs for retired packages on [Hex.pm](https://hex.pm/)
or for archived repositories within the specified GitHub organizations.
